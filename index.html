HTML

  • The head element contains information about the webpage and helps with SEO (Search Engine Optimization
  • The h1 tag is the main header of the webpage, followed by h2,h3 etc.. for sub headings and titles
  • The body element represents the visible content shown to the user
  • The img element displays images with "src=(url)" and "alt=(text information about the image)"
  • This information is written in a "section" element that organizes text into sections

GIT

The GIT Flow

  1. Start by navigating to the correct directory by using cd (directory name)
  2. Open directory in VS using code .
  3. Double check VS to make sure the correct files are present
  4. Check current branch by using git status
    • Make sure you are in the MAIN Branch
    • Create a new FEATURE branch using git checkout -b feature/(branch name)
  5. Use git status to verify you are in the newly created branchh
  6. Use git add -A to add the code to the working branch
  7. Now that the files are staged use git commit -m(info message) to commit the files
  8. Before pushing to the REPO make sure the local branch is in sync with the base branch using git pull origin main
  9. Push changes using git push origin (branch name)
  10. Verify in GITHUB, create pull request, review, and merge to main
